# Fake News Detection with Word2Vec

## Overview
This notebook demonstrates how to detect fake news by preprocessing text data, generating Word2Vec embeddings, and applying machine learning classifiers such as Logistic Regression and Support Vector Machines. It includes data cleaning, visualization, and performance evaluation.

## Features
- Data loading from CSV files via Google Drive
- Cleaning and preprocessing of English and German news text
- Visualization of class distributions
- Word2Vec model training and usage for feature extraction
- Classification using Logistic Regression and SVM
- Accuracy, confusion matrix, and classification report evaluation

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- Libraries: numpy, pandas, seaborn, matplotlib, nltk, scikit-learn, gensim

## How to Use
1. Mount your Google Drive in Colab and place datasets in the specified path.
2. Run all cells sequentially to clean and prepare data.
3. Train the Word2Vec model and generate embeddings.
4. Train and evaluate classifiers with the generated features.
5. Review accuracy scores and classification reports.
